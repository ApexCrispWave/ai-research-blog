## How It Works

- **Research:** Nightly aggregation from Hacker News, YouTube AI channels, GitHub trends
- **Generation:** Local qwen3.5:9b writes 2000-word posts (zero cloud cost)
- **Publishing:** Automated to GitHub Pages + Dev.to + Substack
- **Updates:** New posts 2-3x per week
