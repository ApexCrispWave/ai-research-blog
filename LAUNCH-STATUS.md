## 🎓 How to Use This System

### For Manual Publishing

```bash
# 1. Generate research
bash scripts/aggregate-research.sh

# 2. Create blog post
bash scripts/generate-blog-post.sh

# 3. Review post
cat posts/*.md | less

# 4. Publish
bash scripts/publish-blog-post.sh posts/*.md
```

### For Automated Publishing (Cron)

```bash
# Install cron jobs
bash scripts/setup-cron.sh

# Monitor logs
tail -f logs/cron-research.log
tail -f logs/cron-blog-gen.log
```

### To Customize Blog

Edit these files:
- **Title/Description:** `_config.yml`
- **Homepage:** `index.md`
- **Post Layout:** `_layouts/post.html`
- **Research Focus:** `scripts/aggregate-research.sh` (modify prompts)
- **Blog Generation:** `scripts/generate-blog-post.sh` (modify prompts)
